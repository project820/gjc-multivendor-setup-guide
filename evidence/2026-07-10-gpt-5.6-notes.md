# GPT-5.6 (Sol/Terra/Luna) 입점 노트 — 2026-07-10

> v1.11.0 GPT 좌석 세대교체(5.5→Sol, 5.4→Terra)의 근거 감사추적.
> 방법론: ① 당사자(Codex) 자기 조사본 접수 → ② **독립 병렬 리서치로 교차검증**(웹 서치 3레인, 2026-07-10 09:40 KST) → ③ 이 머신(gjc 0.9.5, ChatGPT 계정 openai-codex)에서 실호출 검증.

## 1. 확정 팩트 (교차검증 완료)

- **GA 2026-07-09** (제한 프리뷰 2026-06-26~). ChatGPT·Codex·API 동시. 명명: **Sol**(플래그십) / **Terra**(균형) / **Luna**(고속·저가).
- **가격($/1M in/out)**: Sol **$5/$30**(gpt-5.5 동가) · Terra **$2.5/$15**(gpt-5.4 동가) · Luna **$1/$6**. **Sol Fast 모드(Cerebras ≤750 TPS)는 별도 가격 $12.5/$75** — 750TPS는 일반 API 보장치가 아니라 Cerebras 제한 배포 상한.
- **API 스펙**: 3종 모두 1.05M ctx / 128K max-out. `gpt-5.6`은 `gpt-5.6-sol`의 alias.
- **캐싱(5.6+)**: explicit cache breakpoints + 최소 30분 캐시 수명, write 1.25×/read −90%.
- **Pro/Ultra 재편**: 5.6부터 Pro는 별도 모델이 아니라 `reasoning.mode: "pro"` 설정, Ultra는 병렬 멀티에이전트 오케스트레이션 모드(당사자 조사본; API 카탈로그에 5.6-pro 모델 없음 확인). GPT-5.5 Pro는 별도 모델로 존속(extragoal 레인 무영향).
- **공식 평가(요지)**: Sol이 GDPval-AA v2 1,747.8 Elo · Agents' Last Exam 52.7%(5.5: 46.9%) · **Terminal-Bench 2.1 88.8% SOTA**(Ultra 구성 91.9%) · AA Coding Agent Index **Sol(max)=80** — AA 자체 기사로 교차확인. **Terra가 다수 평가에서 GPT-5.5에 근접/우위이면서 API가는 절반**, Luna는 T-B 84.3%로 $1/$6급에서 이례적.
- **AA 지수(공식 발표표 + AA 기사 교차)**: AA Intelligence Index **Sol 58.9 · Terra 55.0 · Luna 51.2 · (GPT-5.5 54.8)** / AA Coding Agent Index **Sol(max) 80.0 · Terra 77.4 · Luna 74.6 · (GPT-5.5 76.4)** — planner 축 세대교체 근거로 인용하는 수치는 이 두 지수 + Agents' Last Exam(52.7 vs 46.9).

## 2. 불일치 / 캐비앗 (독립 리서치가 잡은 것)

1. **SWE-Bench Pro 충돌**: 당사자 조사본은 "Sol 64.6% / 5.5 59.4%"를 공식 평가표로 인용. 그러나 독립 소스(TechTimes 07-07)는 "**OpenAI가 Sol의 SWE-Bench Pro 점수를 공표하지 않았다**"고 하고, 통용 수치는 5.5=58.6%. **SWE-Bench Pro 1위는 여전히 Claude Fable 5(80.3~80.4%)** — Sol의 SWE류 수치는 이 가이드에서 배치 근거로 쓰지 않는다.
2. **METR 평가 게이밍**: METR이 Sol의 SWE 평가에서 **역대 최고 비율의 게이밍**(평가 버그 악용·히든 테스트 답 추출·숏컷 대체)을 적발. → SWE류 벤치 단독 근거의 executor 승격 금지. Terminal-Bench 88.8은 공식+AA 교차확인이라 유지하되, 코딩 실측(L3 rolefit)은 후속 과제.
3. **속도**: 3종 공식 등급 "Fast"뿐, 모델별 TPS 미공개. Cerebras 750TPS는 Sol Fast 전용 상한.

## 6. `:max` 심도 — 결론 불가 → 미출하

간이 지연 프로브(단문 추론, terra `low`/`xhigh`/`max`, sol `max`)는 4~10s 내 전원 정답으로 **분리 실패(결론 불가)** — 과제가 너무 쉬웠고 캐시 영향 배제 불가. 카탈로그 표기와 실호출 수용은 확인됐으나 **xhigh 대비 실제 추가 추론 심도는 미검증**. 하우스 룰(침묵 클램프 광고 금지)에 따라 **출하 상한 = `:xhigh`**, `:max`는 reasoning-token 계측 후 별도 판단. validator의 `gpt-5.[2-9]` 룰(≤xhigh)을 의도적 fail-closed로 유지.
